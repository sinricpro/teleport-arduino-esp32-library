# Webserver Example

This example demonstrates how to use a WebServer

1. Change the credentials and flash. 
2. Endopint is displayed in Arduino Serial Monitor.
3. Goto HTTP : https://your-endpoint.sinric.tel.
4. Home page is displayed.
